# 《别让AI替代你》美术风格锁定（M2 样张 v1）

> 目的：用 4 张跨类别样张锁定"扁平卡通办公风"基准调性，确认后据此批量产出全部素材。
> 样张位置：`assets/art/style-lock/`

## 1. 锁定的基础风格 prompt（所有后续生图必须含此段）

```
Flat cartoon vector game asset, bold clean outlines, minimal flat color,
no gradients, no photorealism, Dyson Sphere Program / casual WeChat
mini-game art direction, office workplace comedy theme, high readability.
```

- 统一底：样张期用 `isolated on light gray studio background` 便于横向比对调性；
  **正式批量改为 `transparent background, PNG with alpha`**（Cocos sprite 必需）。
- 尺寸：单图标/表情 `1024×1024`，命中特效 `1024×1024` 或 `1024×1536`（竖向爆发）。
- 风格：`natural`（不走高饱和 AI 味）。

## 1.5 备选风格：手绘插画风（对比用）

基础 prompt（替换 §1 第一段）：
```
Hand-drawn illustration game asset, ink-and-watercolor storybook sketch style,
warm paper texture, expressive brush lines, soft muted palette,
office comedy theme, charming and detailed, not flat vector, not 3D render.
```
样张位置：`assets/art/style-lock/handdrawn/`（卡牌/道具/Boss表情/命中特效各一张）。
特点：有质感、有温度，但 AI 跨批一致性最差、包体最大、序列帧最难点 —— 若选此方向需更强人工收尾。

## 3. 配色铁律（来自核心策划，生图务必遵守）

- 卡片类别色互不重复，且**全部避开"返工红"**：
  常规=蓝 / 汇报=橙 / 关键=紫 / 提案=青 / 紧急=琥珀 / 会议·文档=灰 / Boss临检=深黑框。
- 危险区红只出现在 UI（认可度条），场上不出现"活的红色威胁卡"。
- 命中特效可用琥珀+青点缀，不参与类别色体系。

## 4. 命名规范（批量时强制）

```
assets/art/
  cards/      card-routine / card-report / card-key / card-proposal
              card-urgent / card-meeting / card-document / card-boss
  props/      prop-add-demand / prop-change-demand / prop-throw-pot / prop-kiss-up
  boss/       boss-neutral / boss-glance / boss-busy / boss-frown
              boss-annoyed / boss-facepalm / boss-confident / boss-sweat
              boss-terrified / boss-caught (被谈话定格)
  fx/         fx-hit / fx-rework-x / fx-clear-burst / fx-freeze
              fx-combo-star / fx-perfect-glow
```

## 5. Cocos 接入约定（生图后由程序/美术收尾）

1. 透明 PNG 入 `assets/art/*`，Cocos 自动生成 `.meta`；按需建 SpriteFrame（九宫/裁剪）。
2. 表情动画：先出 keyframe（上表 boss/*），再在 Cocos 用 帧动画/补间 串成序列。
3. 特效：fx 作序列帧或粒子贴图，走对象池（低端机性能，见开发计划风险表）。
4. 合图集（TexturePacker/Auto Atlas）控制主包 <4MB（微信小游戏限制）。
